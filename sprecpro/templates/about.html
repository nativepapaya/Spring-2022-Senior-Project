Features

This application utilizes the Spotify API and has the ability to display custom data about songs, albums and profiles.
Users are able to log into this application through their own Spotify account.
User profile data is imported from Spotify, including their top songs. Users can also set a custom SPRecPro profile, bio, age and featured song.
Interactive favorite system that gives users the ability to add songs directly to their profile.
A fully functional comment and like system, giving users the ability to communicate with each other and discuss song reccomendations.
Full Post CRUD. Create, Read, Update and Delete Posts. Posts can also feature a song and its album.

Technology


This application utilizes the following:
Python's MVC framework Django
HTML, CSS, Bootstrap and Javascript
SQLite for the database
Github for Repo Hosting
